# Spring 2026 mini-exploit portfolio

The best, and hardest, class I've taken at UVA was CS 4630, "Defense Against Dark Arts". I got my first real taste of Ghidra, GDB, and objdump (all in a Linux environment). We mainly focused on understanding and implementing low-level memory corruption classes: 
* Tricky jump patch
* Lex tricky jump scanner
* Reverse engineering a stripped executable
* Reverse engineering a Tigress-transformed executable
* Reverse engineering an "encrypted" executable
* Buffer overflow attack
* Subterfuge buffer overflow attack
* Return-oriented-programming chain attack (my favorite!)
* Use-after-free attack
* Rewriting vulnerable Rust code into safe Rust code 
This repository showcases the work I did in implementing each of these exploits/defenses.
